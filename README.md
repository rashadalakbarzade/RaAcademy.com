# RaTrading Academy — Website

## 🏆 Проект

Профессиональный одностраничный сайт-портфолио для трейдера и онлайн-академии **RaTrading Academy** Рашада Алекберзаде.

**Слоган:** «Торговля Против Толпы»  
**Специализация:** Smart Money Concepts (SMC) и Price Action

## ✅ Реализованные функции

### Разделы сайта:
- **Шапка/Навигация** — фиксированный navbar с логотипом, меню, иконками соцсетей и CTA-кнопкой
- **Hero Section** — анимированный фон с графиками, заголовок, статистика, кнопки CTA
- **Бегущая строка** — тикер с торговыми результатами и ключевыми терминами
- **Обо мне** — биография Рашада, профессиональный портрет-placeholder, статистика
- **Программа курса** — 4 блока × 18 уроков с иконками и прогресс-барами
- **Преимущества** — 8 карточек с описанием включённых материалов
- **Roadmap** — 5 этапов развития трейдера на временной шкале
- **Тарифы** — групповое и индивидуальное обучение с сравнением
- **Бонусы** — PDF чек-лист, калькулятор риска, Discord сообщество
- **Кейсы/Отзывы** — Win Rate диаграмма + слайдер с 4 реальными кейсами
- **FAQ** — 5 вопросов с аккордеон-ответами
- **Финальный CTA** — призыв к действию
- **Футер** — полная информация, соцсети, навигация, контакты

- ✅ Адаптивный дизайн (мобильный, планшет, десктоп)
